## Overview

Welcome to **The Wild Oasis**! This is the **customer-facing version** of a hotel management web app where users can:

- View and browse available cabins with images and descriptions.
- Book cabins based on availability and select the number of guests.
- Sign in with Google to manage bookings.
- View, edit, or cancel existing bookings.
- Update profile information.
- Enjoy a fully responsive experience on all devices.

I also built an **admin version** for hotel staff to manage cabins, bookings, and guests.

## Features

- Browse all available cabins with descriptions and images.
- Book a cabin with available dates and guest count.
- Google sign-in with NextAuth.js.
- View, edit, or cancel existing bookings.
- Update user profile.
---

## Technologies Used

- **Next.js** (App Router & Server Components)  
- **Tailwind CSS** for styling  
- **Supabase** as the backend database  
- **NextAuth.js** for authentication (Google sign-in)  
- **date-fns** for date handling
